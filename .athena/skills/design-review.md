# 설계 검토 스킬

## 설명

설계 문서를 검토하고 Phase 구현 계획을 세웁니다. Architect 모드에서 실행합니다.

---

## 작업 단계

### 1. 설계 문서 읽기

다음 설계 문서들을 읽어 프로젝트 구조를 파악합니다:

- [`docs/PRD.md`](docs/PRD.md:1) - 제품 요구사항
- [`docs/STRUCTURE.md`](docs/STRUCTURE.md:1) - 프로젝트 구조
- [`docs/UI.md`](docs/UI.md:1) - UI 가이드라인
- [`docs/LOGIC.md`](docs/LOGIC.md:1) - 비즈니스 로직

### 2. Phase 구현 계획 수립

[Phase 번호]에 대한 구현 계획을 수립합니다.

#### 작업 순서

### 3. Todo List 생성

구현 계획을 바탕으로 상세한 Todo List를 작성합니다.
