# 에러 해결 스킬

## 설명

다양한 에러 상황을 해결합니다.

---

## 에러 유형별 해결 방법

### 1. 터미널 에러

**관련 파일**
- [`app.py`](app.py:1)

**작업**
아래 에러가 발생했을 때 해결합니다. 다른 코드는 건드리지 않습니다.

```
[터미널 에러 메시지 전체 붙여넣기]
```

---

### 2. 브라우저 콘솔 에러

**관련 파일**
- [`static/main.js`](static/main.js:1)

**작업**
브라우저 콘솔에서 에러가 발생했을 때 해결합니다. 다른 코드는 건드리지 않습니다.

```
[콘솔 에러 메시지 전체 붙여넣기]
```

---

### 3. 노션 API 에러

**관련 파일**
- [`app.py`](app.py:1)

**작업**
노션 API 호출 시 에러가 발생했을 때 해결합니다. [`.env`](.env:1) 파일의 키/ID 값은 정상이라고 가정하고, API 호출 코드만 확인하고 수정합니다.

```
[에러 메시지 붙여넣기]
```

---

### 4. 화면에 데이터가 안 나올 때

**관련 파일**
- [`static/main.js`](static/main.js:1)
- [`app.py`](app.py:1)

**작업**
`/api/[엔드포인트]`는 200 OK로 응답하지만 화면에 데이터가 표시되지 않을 때 해결합니다.

**해결 단계**
1. API 응답 JSON 구조와 렌더링 함수의 필드명이 일치하는지 확인
2. 문제 원인 설명
3. 수정

다른 코드는 건드리지 않습니다.
