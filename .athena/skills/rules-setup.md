# Rules 설정 스킬

## 설명

프로젝트 규칙 파일을 생성하여 Athena 시스템에 등록합니다.

## 사용법

이 스킬은 프로젝트 시작 시 한 번만 실행하세요.

---

## 작업 단계

### 1. RULES.md 읽기

[`docs/RULES.md`](docs/RULES.md:1) 파일을 읽어 프로젝트 규칙을 파악합니다.

### 2. 규칙 파일 생성

읽은 내용을 바탕으로 다음 조건으로 규칙 파일을 생성합니다:

- **적용 범위**: 모든 모드 (architect, code, ask)
- **파일 형식**: 하나의 통합 파일
- **파일명**: `ax-sejong-rules.md`
- **저장 위치**: `.athena/rules/`

### 3. 규칙 구성 요소

생성된 규칙 파일은 다음 구조를 포함해야 합니다:

1. **금지 사항** (R-01 ~ R-10)
2. **필수 사항** (R-11 ~ R-20)
3. **Phase 보호 규칙** (R-21 ~ R-26)
4. **코드 스타일 규칙**
5. **파일 구조 규칙**
6. **노션 API 규칙**
7. **UI 규칙**
8. **보안 규칙**
9. **모드별 가이드라인**
