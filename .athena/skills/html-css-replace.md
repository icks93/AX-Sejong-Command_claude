# HTML/CSS 파일 교체 스킬

## 설명

HTML과 CSS 파일을 새로운 버전으로 교체합니다. [`app.py`](app.py:1)와 [`main.js`](static/main.js:1)는 절대 건드리지 않습니다.

---

## 작업 단계

### 1. 현재 파일 읽기

- [`templates/index.html`](templates/index.html:1)
- [`static/style.css`](static/style.css:1)

### 2. 새 파일로 교체

아래 내용을 사용하여 각 파일을 교체합니다.

#### index.html

```
[새로운 HTML 내용 붙여넣기]
```

#### style.css

```
[새로운 CSS 내용 붙여넣기]
```

### 3. 교체 작업

- `templates/index.html`: 새로운 HTML 내용으로 전체 교체
- `static/style.css`: 새로운 CSS 내용으로 전체 교체
