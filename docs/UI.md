## 7. 마우스오버 툴팁

### 동작
- 사업명 텍스트에 점선 밑줄 표시 (`border-bottom: 1px dashed #CBD5E1`)
- 마우스 올리면 사업명 아래 팝업 표시
- 마우스 내리면 즉시 사라짐

### 툴팁 스타일
```
배경: #FFFFFF
테두리: 0.5px solid #CBD5E1
모서리: 8px
패딩: 12px 14px
너비: 300px
위치: 사업명 셀 기준 아래쪽 (top: calc(100% + 6px))
z-index: 99
```

### 툴팁 내용 구조
```
┌────────────────────────────────────────┐
│ [사업명 전체 — 11px/500/#0F172A]       │
│                                        │
│ 고객사    [값 — 10px/#475569]          │
│ 운영대상  [값 — 10px/#475569]          │
│ 요약      [값 — 10px/#475569]          │
└────────────────────────────────────────┘
```

| 항목 | 스타일 |
|------|--------|
| 제목 | 11px / 500 / #0F172A / margin-bottom 6px |
| 레이블 | 10px / #0F172A / min-width 52px |
| 값 | 10px / #475569 / line-height 1.5 |
| 행 간격 | margin-bottom 4px |
