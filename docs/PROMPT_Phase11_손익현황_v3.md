## 작업 목록 (4개)

### 1. %p 표시를 전부 효율배수(x)로 교체 — 영향 범위가 가장 큼

지금까지 "▲7.6억(2.6%p)" 형태로 보여주던 모든 곳을 "▲7.6억(1.21x)" 형태로 바꾼다.

- **계산식**: 효율배수 = `해당 지표 비중(%) ÷ 인력 비중(%)`. 예: 매출 효율배수 = `매출비중_수행조직대비 ÷ 인력비중_수행조직대비`. 매출원가·매출이익도 동일 패턴(분모는 항상 인력비중)
- **노션 컬럼 추가 불필요** — 이미 존재하는 `매출비중_수행조직대비`, `인력비중_수행조직대비` 등을 가져와서 나누기만 하면 됨
- 적용 위치: 미니차트 4개(매출/매출원가/매출이익) 세종 막대 옆 초과분 2번째 줄, 요약표의 "수행인력대비 매출/매출원가/매출이익" 셀
- 2025년(수행조직대비 탭) 매출이익만 예외: 효율배수 산정 불가하므로 "(흑자전환 기여)" 텍스트 유지 (금액 ▲8.4억은 그대로 표시)
- **코드에서 "%p" 문자열을 검색해서 빠짐없이 교체했는지 확인할 것**

### 2. 연도별 효율 카드 — 효율배수 2개로 분리

기존: 카드 상단에 "효율배수"(매출이익 기준) 숫자 1개.
변경: **"매출 효율배수"와 "매출이익 효율배수" 2개**를 나란히 작은 박스로 표시.
- 매출 효율배수 = `매출비중_수행조직대비 ÷ 인력비중_수행조직대비`
- 매출이익 효율배수 = 기존과 동일 계산, 위치만 카드 상단 작은 박스로 이동
- 카드 하단의 인력/매출/매출원가/매출이익 미니바 4개는 변경 없음
- 2025년은 매출 효율배수는 정상 숫자, 매출이익 효율배수만 "흑자전환 기여" 텍스트 (한 카드에 숫자/텍스트가 같이 있을 수 있음)

### 3. KPI 타일 6개 → 8개로 확장

레이아웃을 4열×2행으로 다시 짠다:
- 1행(2026년 기준): 매출 효율배수 / 초과 매출 / 매출이익 효율배수 / 초과 매출이익
- 2행(누적 2021~2026): 평균 매출 효율배수 / 총초과 매출 / 평균 매출이익 효율배수 / 총초과 매출이익
- 색 테마는 2개로 단순화: 매출 계열(시안), 매출이익 계열(핑크) — 기존 "효율배수 골드" 계열은 없어짐
- 평균 매출이익 효율배수는 흑자전환 기여로 처리된 연도(2025) 제외하고 평균, 캡션에 "(○○ 적자해 제외)" 표기

### 4. KPI 영역 디자인 — 더 눈에 띄게

- KPI 타일을 감싸던 패널 박스(테두리+배경)를 제거하고, 페이지 배경에 타일이 바로 노출되게 한다
- 각 타일 상단에 4px 색상 스트라이프 추가 (컬러풀 모드일 때 그라디언트 색, 기본 모드일 때 중립 회색)
- 타일에 약한 box-shadow, 숫자 폰트 크기 키움(24px → 28px)
- 컬러풀 토글(`body.colorful`) 구조는 그대로 유지
