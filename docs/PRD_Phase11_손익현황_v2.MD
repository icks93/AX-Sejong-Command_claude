### 2.3 예외 처리 — 2025년 (수행조직_매출이익 < 0) **v2.0 변경**

2025년은 수행조직 전체가 매출이익 적자(-8.45억)였는데 세종은 흑자(7.53억)였던 해. 영향 범위는 **수행조직대비 탭에서만** 발생한다 (전사_매출이익은 2025년에도 양수이므로 전사대비 탭은 이 예외와 무관).

**비중(%)·효율배수 — 계속 산정 불가, "흑자전환" 처리:**
- `매출이익비중_수행조직대비` 값 자체가 의미 없음(분모가 음수)
- 효율배수(`매출이익비중 ÷ 인력비중`)도 산정 불가
- 화면 표시: 연도별 효율 카드의 효율배수 자리 → **"흑자전환 기여"** 배지. 미니차트(매출이익)의 세종 막대 라벨 비율 부분 → **"(흑자)"**

**초과분 금액(`수행인력대비_매출이익_금액`) — v2.0에서 산정 가능으로 변경:**
초과분 "금액"은 `세종_매출이익 − (수행조직_매출이익 × 인력비중_수행조직대비 ÷ 100)` 으로 계산하며, 이 공식은 수행조직_매출이익이 음수여도 그대로 성립한다 (음수를 빼는 형태가 되어 오히려 초과분이 커짐). **2025년 값 = 8.4억** (= 7.53 − (−8.45 × 0.1076)).
다만 `수행인력대비_매출이익_pp`(%p)는 `매출이익비중_수행조직대비`가 정의되지 않으므로 **계속 산정 불가**.

→ **화면 표시 규칙**: 금액(▲8.4억)은 정상 표시하고, %p 자리만 "(흑자전환 기여)" 텍스트로 대체한다. (v1.0에서는 금액까지 통째로 "흑자전환 기여"로 뭉뚱그렸는데, v2.0은 금액과 %p를 분리해서 처리)

판단 조건:
- 비중/효율배수: `해당연도.수행조직_매출이익 < 0` → 흑자전환 배지
- 초과분 금액: 항상 계산 (음수 분모 문제 없음)
- 초과분 %p: `해당연도.수행조직_매출이익 < 0` → "(흑자전환 기여)" 텍스트로 대체

### 3.1 KPI 타일 (v2.0 신규)

가로 6개 타일, 3가지 색 테마:
- **효율배수 계열(골드)**: 2026년 효율배수, 누적(2021~2026) 평균 효율배수
- **초과매출 계열(시안)**: 2026년 초과 매출, 누적 총초과 매출
- **초과매출이익 계열(핑크)**: 2026년 초과 매출이익, 누적 총초과 매출이익

계산 정의:
- `초과 매출/매출이익` = 해당 연도의 `수행인력대비_매출_금액` / `수행인력대비_매출이익_금액` (탭에 따라 수행조직 또는 전사 기준 컬럼)
- `평균 효율배수` = 효율배수가 산정 가능한 연도(흑자전환 기여로 처리된 연도 제외)의 평균. 제외된 연도가 있으면 캡션에 "(○○ 적자해 제외)" 표기
- `총초과 매출/매출이익` = 2021~2026 전체 연도의 초과분 금액 합계. 매출이익은 2.3절에 따라 2025년도 포함하여 합산 가능(수행조직대비 기준)

**디자인 토글**: `body.colorful` 클래스 유무로 컬러풀 모드 on/off. 실제 구현 시에도 이 방식(클래스 하나로 색 on/off)을 유지해서, 추후 익스가 톤다운을 원하면 코드 수정 없이 바로 되돌릴 수 있게 할 것. 다만 실제 배포 시 토글 버튼 자체는 사용자 노출용으로 유지할지, 개발자 전용으로 숨길지는 별도 논의.

### 3.4 하이라이트 배너 — **탭별로 메시지 전체가 다름**

**수행조직대비 탭** (기존 v1.0과 동일):
> "2025년 — 수행조직이 적자였던 해"
> −8.5억(수행조직 매출이익) + 7.5억(세종 매출이익) = 적자(−3.0%) 속 세종 16.7%

**전사대비 탭** (v2.0 신규):
> "2025년 — 전사 매출이익률이 가장 낮았던 해"
> 5.2%(전사 매출이익률, 전체 기간 중 최저) vs 16.7%(세종 매출이익률)
> "회사 전체 매출이익률이 가장 낮았던 2025년에도, 세종개발본부는 16.7%의 매출이익률을 유지했습니다. 회사 평균보다 3배 이상 높은 수치입니다."

자동 판단 로직(확장 여지, 1차는 2025 고정 가능):
- 수행조직대비: `수행조직_매출이익 < 0`인 가장 최근 연도가 있으면 그 연도로, 없으면 배너 숨김
- 전사대비: `전사_매출이익률`이 최저인 연도를 자동 탐색

### 3.5 연도별 사업 현황 요약 표 (기존 4.4절과 동일, 탭 대응 추가)

- 표의 1번째 행(상단) 라벨이 탭에 따라 "수행조직" ↔ "전사"로 전환, 해당 행의 수치도 전환
- 2번째 행(세종)의 "수행인력대비" 3컬럼 계산 기준도 탭에 따라 전환
- 2025년 매출이익 셀(수행조직대비 탭일 때만): **"▲8.4억 (흑자전환 기여)"** 형태로 표시 — v1.0의 "흑자전환 기여" 단독 표기에서 변경

## 4. API 연동 방식 (기존 PRD 5절과 동일, 변경 없음)

1. FastAPI 백엔드에 `/api/profit-loss` 엔드포인트
2. Notion API로 `[AX]세종손익현황_연도별` 6행 조회
3. Formula 속성값은 그대로 사용, 재계산 금지
4. 프론트(Vanilla JS)가 3절 블록에 매핑. **탭 전환은 프론트에서만 처리** (동일 API 응답 안에 수행조직/전사 컬럼이 모두 들어있으므로 탭 전환 시 재호출 불필요)
5. 필드명은 노션 속성명 그대로 사용

## 6. 비기능 요구사항 (기존과 동일)

- 반응형: KPI 타일은 데스크탑 6열 → 태블릿 3열 → 모바일 2열
- 숫자 포맷: 인력=정수+"명", 금액=소수1자리+"억", 비율=소수1자리+"%", %p=소수1자리+"%p"
- 색상: 수행조직/전사=스틸 계열(공통), 세종=골드, 증가=레드(▲)/감소=블루(▼), KPI 타일은 골드/시안/핑크 3색 (컬러풀 모드 시)
