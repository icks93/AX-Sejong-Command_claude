> ⚠️ 노션 Formula는 **다른 Formula 속성을 참조하지 못한다.** 모든 수식은 원본 Number 컬럼(전사_*/수행조직_*/세종_*)에서 직접 계산하도록 작성되어 있음. 추가 Formula 컬럼이 필요할 때도 이 제약을 지킬 것.

> **v3.0 변경**: 노션의 `_pp`(퍼센트포인트) 컬럼들은 **화면에 더 이상 표시하지 않는다.** 대신 화면에는 "효율배수(x)" = `매출비중_수행조직대비 ÷ 인력비중_수행조직대비` (또는 매출원가/매출이익 동일 패턴) 를 표시한다. **이 계산은 이미 존재하는 두 비중 컬럼을 프론트/백엔드에서 나누기만 하면 되므로 노션에 컬럼을 추가할 필요가 없다.** `_pp` 컬럼 자체는 노션에 그대로 남겨두되(삭제 불필요), 화면 렌더링에서만 참조를 끊는다.

### 2.3 예외 처리 — 2025년 (수행조직_매출이익 < 0) **v2.0 변경**

2025년은 수행조직 전체가 매출이익 적자(-8.45억)였는데 세종은 흑자(7.53억)였던 해. 영향 범위는 **수행조직대비 탭에서만** 발생한다 (전사_매출이익은 2025년에도 양수이므로 전사대비 탭은 이 예외와 무관).

**비중(%)·효율배수 — 계속 산정 불가, "흑자전환" 처리:**
- `매출이익비중_수행조직대비` 값 자체가 의미 없음(분모가 음수)
- 효율배수(`매출이익비중 ÷ 인력비중`)도 산정 불가
- 화면 표시: 연도별 효율 카드의 효율배수 자리 → **"흑자전환 기여"** 배지. 미니차트(매출이익)의 세종 막대 라벨 비율 부분 → **"(흑자)"**

**초과분 금액(`수행인력대비_매출이익_금액`) — v2.0에서 산정 가능으로 변경:**
초과분 "금액"은 `세종_매출이익 − (수행조직_매출이익 × 인력비중_수행조직대비 ÷ 100)` 으로 계산하며, 이 공식은 수행조직_매출이익이 음수여도 그대로 성립한다 (음수를 빼는 형태가 되어 오히려 초과분이 커짐). **2025년 값 = 8.4억** (= 7.53 − (−8.45 × 0.1076)).
다만 `매출이익비중_수행조직대비`가 정의되지 않으므로 효율배수(매출이익비중÷인력비중)도 **계속 산정 불가**.

→ **화면 표시 규칙**: 금액(▲8.4억)은 정상 표시하고, 효율배수 자리만 "(흑자전환 기여)" 텍스트로 대체한다.

판단 조건:
- 비중/효율배수: `해당연도.수행조직_매출이익 < 0` → 흑자전환 배지
- 초과분 금액: 항상 계산 (음수 분모 문제 없음)
- 초과분 효율배수: `해당연도.수행조직_매출이익 < 0` → "(흑자전환 기여)" 텍스트로 대체

### 3.1 KPI 타일 — **v3.0 6개→8개로 확장**

가로 4열 × 2행(2026년 기준 / 누적), 2가지 색 테마로 단순화:
- **매출 계열(시안)**: 매출 효율배수, 초과 매출
- **매출이익 계열(핑크)**: 매출이익 효율배수, 초과 매출이익

(v2.0의 "효율배수(골드)" 단일 계열은 폐지하고, 매출/매출이익 각각의 효율배수로 분리했다.)

계산 정의:
- `매출 효율배수` = `매출비중_수행조직대비 ÷ 인력비중_수행조직대비` (연도별 미니차트의 효율배수와 동일 계산, 3.3절 참고)
- `매출이익 효율배수` = `매출이익비중_수행조직대비 ÷ 인력비중_수행조직대비`. **2025년(수행조직대비 탭)은 산정 불가 → 해당 연도는 평균 계산에서 제외**
- `초과 매출/매출이익` = 해당 연도의 `수행인력대비_매출_금액` / `수행인력대비_매출이익_금액`
- `평균 매출 효율배수` = 2021~2026 전체 평균 (매출은 예외 연도 없음)
- `평균 매출이익 효율배수` = 효율배수가 산정 가능한 연도의 평균 (흑자전환 기여 연도 제외, 캡션에 "(○○ 적자해 제외)" 표기)
- `총초과 매출/매출이익` = 2021~2026 전체 연도의 초과분 금액 합계 (2.3절에 따라 2025년 매출이익도 포함하여 합산 가능)

**디자인 — v3.0 강조 처리**:
- 패널 박스(테두리·배경 패널)를 제거하고 페이지 배경에 타일이 직접 노출되도록 한다
- 각 타일 상단에 4px 색상 스트라이프(컬러풀 모드일 때 그라디언트, 기본 모드일 때 중립색)
- box-shadow로 타일에 약한 입체감 부여, 숫자 폰트 크기 확대(24px→28px)
- 목적: 기존 KPI 타일이 "눈에 잘 안 보인다"는 피드백 반영 — 대시보드(Phase 10) KPI 박스처럼 한눈에 들어오게

**컬러풀 토글**: `body.colorful` 클래스 유무로 on/off, v2.0과 동일한 구조 유지 (코드 수정 없이 디자인 원복 가능하게 하는 목적).

### 3.2 연도별 수행인력 효율배수 카드 — **v3.0 변경**

기존(v1.0~v2.0)에는 카드 상단에 "효율배수"(매출이익 기준) 숫자 1개만 크게 표시했다. **v3.0부터는 2개로 분리 표시**:

- **매출 효율배수** = `매출비중_수행조직대비 ÷ 인력비중_수행조직대비`
- **매출이익 효율배수** = `매출이익비중_수행조직대비 ÷ 인력비중_수행조직대비` (기존과 동일 계산, 위치만 이동)

두 숫자를 카드 상단에 가로로 나란히 배치(작은 박스 2개), 그 아래 기존 4개 미니바(인력/매출/매출원가/매출이익, 인력비중 기준선 포함)는 변경 없이 유지.

2025년(수행조직대비 탭)은 매출 효율배수는 정상 표시(수행조직_매출은 항상 양수이므로 산정 가능), 매출이익 효율배수만 "흑자전환 기여"로 대체 — **두 숫자가 같은 카드에서 서로 다른 상태(하나는 숫자, 하나는 텍스트)로 보일 수 있음에 유의**.

### 3.3 연도별 [기준]대비 매출·매출원가·매출이익 미니차트 4개 (기존 4.2절과 동일, 탭 대응 추가)

- 막대 색(수행조직/전사 공통 스틸 톤, 세종 골드)은 유지, 라벨 텍스트만 "수행조직" ↔ "전사" 전환
- 인력비중 기준선·초과분 표시 로직은 동일하되, **초과분 2번째 줄은 %p가 아니라 효율배수(x)로 표시** (v3.0 변경, 2.3절 참고)

### 3.4 하이라이트 배너 — **탭별로 메시지 전체가 다름**

**수행조직대비 탭** (기존 v1.0과 동일):
> "2025년 — 수행조직이 적자였던 해"
> −8.5억(수행조직 매출이익) + 7.5억(세종 매출이익) = 적자(−3.0%) 속 세종 16.7%

**전사대비 탭** (v2.0 신규):
> "2025년 — 전사 매출이익률이 가장 낮았던 해"
> 5.2%(전사 매출이익률, 전체 기간 중 최저) vs 16.7%(세종 매출이익률)
> "회사 전체 매출이익률이 가장 낮았던 2025년에도, 세종개발본부는 16.7%의 매출이익률을 유지했습니다. 회사 평균보다 3배 이상 높은 수치입니다."

자동 판단 로직(확장 여지, 1차는 2025 고정 가능):
- 수행조직대비: `수행조직_매출이익 < 0`인 가장 최근 연도가 있으면 그 연도로, 없으면 배너 숨김
- 전사대비: `전사_매출이익률`이 최저인 연도를 자동 탐색

### 3.5 연도별 사업 현황 요약 표 (기존과 동일, 탭 대응 추가)

- 표의 1번째 행(상단) 라벨이 탭에 따라 "수행조직" ↔ "전사"로 전환, 해당 행의 수치도 전환
- 2번째 행(세종)의 "수행인력대비" 3컬럼 계산 기준도 탭에 따라 전환
- **각 셀의 2번째 값(괄호 안)은 %p가 아니라 효율배수(x)** — 예: `▲7.6억(1.21x)` (v3.0 변경)
- 2025년 매출이익 셀(수행조직대비 탭일 때만): **"▲8.4억 (흑자전환 기여)"** 형태로 표시 (효율배수 자리만 텍스트 대체, v2.0과 동일 규칙)

## 4. API 연동 방식 (변경 없음)

1. FastAPI 백엔드에 `/api/profit-loss` 엔드포인트
2. Notion API로 `[AX]세종손익현황_연도별` 6행 조회
3. Formula 속성값은 그대로 사용, 재계산 금지. **단, 화면의 "효율배수(x)" 표시는 노션 Formula가 아니라 프론트/백엔드에서 두 비중 컬럼을 나눠서 계산** (2.2절 참고, 노션에 새 컬럼 불필요)
4. 프론트(Vanilla JS)가 3절 블록에 매핑. **탭 전환은 프론트에서만 처리** (동일 API 응답 안에 수행조직/전사 컬럼이 모두 들어있으므로 탭 전환 시 재호출 불필요)
5. 필드명은 노션 속성명 그대로 사용

## 6. 비기능 요구사항

- 반응형: KPI 타일은 데스크탑 4열×2행 → 태블릿 2열×4행
- 숫자 포맷: 인력=정수+"명", 금액=소수1자리+"억", 비율=소수1자리+"%", **효율배수=소수2자리+"x"**
- 색상: 수행조직/전사=스틸 계열(공통), 세종=골드, 증가=레드(▲)/감소=블루(▼), KPI 타일은 매출=시안/매출이익=핑크 2색 (컬러풀 모드 시), 타일 상단 4px 색 스트라이프
