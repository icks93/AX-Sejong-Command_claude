---

## 1. 프로젝트 개요

| 항목 | 내용 |
|------|------|
| 프로젝트명 | AX Sejong Command |
| 목적 | 세종개발본부 업무 현황을 한 화면에서 조회하고 자동화를 실행하는 대시보드 |
| 사용자 | 노승익 본부장 (1인, 추후 팀원 공유 가능) |
| 접속 방식 | localhost (내 PC에서 실행, 추후 팀 공유 가능 구조) |
| 개발 방식 | 바이브코딩 (Claude 설계 → 아테나 코드 + GLM-4.7 개발) |
| Phase 1 목표 | 브라우저에서 세종사업현황 탭이 정상 동작하는 상태 |

## 5. KPI 박스 정의

| 박스 | 데이터 소스 | 계산 방법 |
|------|-----------|---------|
| 총 수행 사업 | 노션 DB 전체 행 수 | COUNT(*) |
| 총 매출액 | 노션 DB 매출액 합계 | SUM(매출액) / 100,000,000 → "X.X억" |
| 운영중 | 상태 = "운영중" 또는 "신규" | COUNT(상태 IN 운영중, 신규) |
| 종료 | 상태 = "종료" | COUNT(상태 = 종료) |

## 6. 테이블 컬럼 정의

| 컬럼 | 노션 속성 | 표시 형식 |
|------|---------|---------|
| 사업명 | 사업명 (title) | 텍스트, 마우스오버 툴팁 |
| 고객사 | 고객사 (text) | 텍스트, 흐린 색상 |
| PM | PM (text) | 텍스트 |
| 매출액 | 매출액 (number) | X.X억 형식 |
| 계약기간 | 계약기간 (text) | ~YY.MM.DD 형식 |
| 상태 | 상태 (select) | 배지 (운영중🟢 / 신규🔵 / 종료⚫) |
