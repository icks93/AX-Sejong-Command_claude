## 공통 파싱 규칙 (Phase 3, 4 동일)

```python
# 월 구분: heading_2 + "[26-" 포함
# 자식 블록 없음 → fetch_block_children() 1회만 호출
# 캐싱: 10분 (Phase 2와 동일)
# 테이블 파싱: block type == "table" → rows 추출
#   각 row의 cells → plain_text 추출

def parse_table(block):
    """table 블록에서 2차원 배열로 추출"""
    rows = []
    for row in block.get("table", {}).get("rows", []):
        cells = []
        for cell in row.get("cells", []):
            text = "".join(t.get("plain_text", "") for t in cell)
            cells.append(text)
        rows.append(cells)
    return rows
```
