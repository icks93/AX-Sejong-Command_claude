현재 프로젝트의 노션 연동 상태를 점검한다.

1. **.env 파일 확인**: `.env` 파일이 존재하는지, `NOTION_API_KEY`와 `NOTION_PROJECTS_DB_ID`가 설정되어 있는지 확인한다. 값은 출력하지 않는다(보안).

2. **app.py 상수 확인**: app.py에서 `_PAGE_ID` 또는 `_DB_ID`로 끝나는 모든 상수를 읽어 현재 연동된 노션 페이지/DB 목록을 표시한다.

3. **API 엔드포인트 목록 출력**: app.py에서 `@app.get("/api/` 패턴을 찾아 등록된 모든 엔드포인트를 나열한다.

4. **서버 실행 여부 확인**: `http://localhost:8001/api/projects` 에 요청을 보내 서버가 실행 중인지 확인한다. 실행 중이면 응답 상태를 알려주고, 실행 중이 아니면 `python app.py` 실행 방법을 안내한다.

5. **Phase별 연동 현황 요약**: Phase 1~9 각각에 대해 노션 ID가 정의되어 있는지, API 엔드포인트가 존재하는지 표로 정리해서 보여준다.
